# pharmacyweb
1) Скачать zip архив
2) Через shell открытый в директории запустить команду **docker-compose down -v**, для очистки баз данных, от других проектов
3) В том же терминале ввести команду **docker-compose up --build**, чтобы начать работу контейнера
4) В случаи возникновения ошибки базы данных, зайти на **172.18.0.1:8081** и самому импортировать файл _/src/apteka.sql_, далее все будет работать
